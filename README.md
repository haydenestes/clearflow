# ClearFlow — Personal Finance

> Upload your bank, credit, and Venmo statements. Get a clean monthly cash flow and spending breakdown — no accountant required.

![Design](https://img.shields.io/badge/Design-Slate%20%2B%20Sage-6b8f71) ![No Backend](https://img.shields.io/badge/Backend-None%20Required-green) ![Privacy](https://img.shields.io/badge/Privacy-100%25%20Local-blue)

---

## What It Does

- **Upload** your bank CSV, credit card CSV, and Venmo CSVs
- **Auto-categorizes** transactions: rent, groceries, dining, travel, gym, subscriptions, and more
- **Generates** a monthly cash flow statement and P&L
- **Exports** full transaction list as CSV

## Supported Sources

| Source | Format |
|---|---|
| US Bank checking | CSV export |
| Citi credit card | CSV export |
| Venmo | Account statement CSV |
| Any bank | Standard date/description/amount CSV |

## Privacy

- 100% browser-based — no server, no database, no account
- Your financial data never leaves your device
- No cookies, no tracking

## Getting Started

```bash
git clone https://github.com/haydenestes/clearflow.git
cd clearflow
open index.html
```

Or deploy to GitHub Pages: Settings → Pages → Source: `main`.

## Design

Slate + Sage color system. Fraunces serif + Nunito Sans. Built to be clean and approachable for non-finance people.
